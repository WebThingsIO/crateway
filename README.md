## Getting started
* [Install rustup](https://rustup.rs/)
* `rustup default nightly`
* Install dependencies:
  * `sudo apt-get install libsqlite3-dev luarocks`
  * `sudo luarocks install lsqlite3`
* `./setup-ui.sh`
* `cargo run`
* open http://localhost:8000
